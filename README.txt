Dumathoin (doo-muh-thoe-in) 

The purpose of this project is to extract data from the D&D player handbook. 
This data will be used in a database to generate loot tables and shop inventories. 

This is project is dependant on the html file from the Equipment chapter of the Player Handbook on D&D Beyond.
URL: https://www.dndbeyond.com/sources/dnd/phb-2024/equipment#Equipment
